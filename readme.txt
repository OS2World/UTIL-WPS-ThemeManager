Needed tools:
-------------
rc.exe v. 4.00.x
IBM visual age C compiler v. 3.0.8


To build an up to date version of Theme Manager
copy the files from nls\en to one of the languages
directories, translate them and modify the bitmap
file as needed (see the example in the IT directory),
then run BUILD.CMD and select the proper language.

Please check with Alex Taylor as he might have
an updated version of appmsgs.h.
